Ark Discord Bot

A simple discord based server controller for ARK: Survival Evolved.

Easy-To-Use Discord Bot

Easily control your local ARK: Survival Evolved server via discord commands.

Built with DiscordJS and NodeJS

Uses Discord hooks to start, stop, and update the server. Only allows commands from approved user roles.

Commands

!start [map]

Start the server with the given map name.

!stop

Stop the server if it is running.

!status

Checks local processes and the Steam API for server status.

!ip

Gives IP address of the server. Useful for DHCP-based ISPs, especially after updates.

!update

Updates the server using SteamCMD, if it is available. Work-in-progress!
